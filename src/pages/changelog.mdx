What's new across the Qeet APIs and this portal. For real-time availability and
incidents, see the [status page](https://status.qeet.in).

## 2026-06-30 — Developer portal relaunch

<span class="pill pill--brand">portal</span>

- Rebuilt **apis.qeet.in** as a full developer portal: branded landing,
  quickstart, authentication, errors and SDK guides.
- New interactive **API Reference** powered by Scalar — global search, a live
  request console, multi-language code samples, and dark mode.
- Every spec is now downloadable for OpenAPI tooling and client generation.

## 2026-06-25 — Qeet ID in production

<span class="pill pill--brand">Qeet ID 0.2.0</span>

- The Qeet ID platform is **live in production** at `api.id.qeet.in`.
- Reference covers the full surface: Authentication, Management, Federation,
  Developer and Operations APIs.

## Earlier — Qeet Notify v1

<span class="pill pill--brand">Qeet Notify 0.1.0</span>

- Multi-channel transactional notifications API — email, SMS, WhatsApp, in-app
  and webhook — available at `api.notify.qeet.in`.
